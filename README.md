# Akan Name Generator

## Description
A web application that calculates the day of the week a user was born and assigns them an Akan name based on their gender. Akan names come from Ghanaian culture, where children are named based on the day they were born.

## Author
John Chris Wamai

## Technologies Used
- HTML
- CSS
- JavaScript

## Setup Instructions
1. Clone the repository:
   git clone git@github.com:johnchriswamai/akan-name-generator.git
2. Open index.html in your browser

## Live Site
https://johnchriswamai.github.io/akan-name-generator/

## BDD (Behavior Driven Development)

| Behavior | Input | Output |
|---|---|---|
| User enters a valid birthdate and selects male | Day: 23, Month: 10, Year: 2007, Male | Your Akan Name is Kwabena |
| User enters a valid birthdate and selects female | Day: 1, Month: 1, Year: 2000, Female | Your Akan Name is Ama |
| User clicks reset | Any filled form | Form clears and result disappears |
